# 测试用例记录

## 优化前测试结果

### Case 1: 犬 - 标准成年犬
- 宠物：犬
- 体重：10kg
- 性别：男宝
- 绝育：已绝育
- 运动：中度运动
- 体型：正常
- 年龄：成年犬
- 孕哺：非孕/哺乳
- 术后：健康

**计算过程：**
- RER = 70 × 10^0.75 = 70 × 5.623 = 393.64
- 系数推导：
  - 绝育犬中度运动: 1.6
  - 正常体型(继承): 1.6
  - 成年犬(继承): 1.6
  - 非孕/哺乳期(继承): 1.6
  - 健康状态(继承): 1.6
- MER = 393.64 × 1.6 = 629.82 ≈ 630 千卡/天

### Case 2: 猫 - 标准成年猫
- 宠物：猫
- 体重：4kg
- 性别：女宝
- 绝育：未绝育
- 户外：纯室内
- 体型：正常
- 年龄：成年猫
- 怀孕：否
- 术后：健康

**计算过程：**
- RER = 70 × 4^0.75 = 70 × 2.828 = 197.99
- 系数推导：
  - 未绝育猫: 1.4
  - 室内(继承): 1.4
  - 正常体重(继承): 1.4
  - 成年猫(继承): 1.4
  - 非孕期(继承): 1.4
  - 健康状态(继承): 1.4
- MER = 197.99 × 1.4 = 277.19 ≈ 277 千卡/天
